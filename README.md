# Machine_Lab

Machine_Lab is a small collection of Python notebooks and scripts used for learning and experimenting with matrix operations, numerical computing, and fundamental programming concepts. It is intended for students and beginners working through hands-on exercises involving NumPy, linear algebra operations, and basic algorithm implementations.

**Contents**
- `Lab_1.ipynb` — Jupyter notebook with examples and exercises (matrix addition, multiplication, NumPy demos).

**Features**
- Matrix addition and multiplication examples
- NumPy usage and vectorized operations
- Small, easy-to-follow exercises for learning Python control flow and functions

**Requirements**
- Python 3.8+ (recommended)
- Jupyter Notebook or JupyterLab
- NumPy

Install dependencies with pip:

```bash
pip install numpy jupyter
```

**Usage**
- Open the notebook in Jupyter:

```bash
jupyter notebook Lab_1.ipynb
```

- Or run specific Python scripts (if present):

```bash
python path/to/script.py
```

**Examples**
- Run the notebook cells in `Lab_1.ipynb` to see matrix operations and results.
